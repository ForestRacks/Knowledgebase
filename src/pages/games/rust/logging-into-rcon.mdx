# Logging into Rcon.io

To login to rcon.io you will need you server ip address,rcon port which is 1 above the one your server is running on and the rcon password which is located in the startup settings.

Just copy all that information into the spaces provided to login to [rcon.io](https://rcon.io) then you have a console that will show your servers frame rate and the ping of your players.

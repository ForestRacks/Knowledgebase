# Installing FastAsyncWorldEdit

FastAsyncWorldEdit is a plugin that allows you to edit large areas of your world without causing lag. It is a fork of the popular WorldEdit plugin.

1. Download the latest version of FastAsyncWorldEdit from [here](https://ci.athion.net/job/FastAsyncWorldEdit/).
2. Log into the [ForestRacks Control Panel](https://panel.forestracks.com).
3. Navigate to your server and click on the `File Manager` tab.
4. Navigate to the `plugins` folder.
5. Click on the `Upload` button and upload the FastAsyncWorldEdit jar file.

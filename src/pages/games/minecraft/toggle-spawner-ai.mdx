# Enable or Disable Spawner Mob AI
Often servers will disable spawner animal or monster artificial intelligence since large spawner farms can cause massive server performance impacts.

<Steps>
### Login to Panel
First login on [https://panel.forestracks.com](https://panel.forestracks.com) and click on your server. If you cannot login, follow the [Troubleshooting Guide](/games/troubleshoot/invalid-login).
### Navigate to Files
Click the "File Manager" tab

### Find Spigot Config
Locate and open the `spigot.yml` config file.

### Change Mob AI Status
Use the panel search function to find `nerf-spawner-mobs` and set it to true if you want to disable spawner mob AI or false to enable spawner mob AI.
<Tabs items={['Enable AI', 'Disable AI']}>
  <Tab>
    ```yml copy
    nerf-spawner-mobs: false
    ```
  </Tab>
  <Tab>
    ```yml copy
    nerf-spawner-mobs: true
    ```
  </Tab>
</Tabs>
</Steps>

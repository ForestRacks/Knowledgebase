# How To Enchant Items In Vanilla Minecraft.

<Steps>
### Admin Permissions
Grant yourself server operator privileges or enable cheats if you are creating the item in single player.

### Give Item
type the following:
```ini copy
/give @s minecraft:stick{Unbreakable:1}
```
Note: The item "stick" can be replaced with other items such as "diamond_chestplate"
</Steps>

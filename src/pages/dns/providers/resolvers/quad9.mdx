# Using Quad9 9.9.9.9 DNS

Cloudflare's Public DNS Resolver is a project created in collaboration between Cloudflare and APNIC. Cloudflare only keeps 24 hours worth of logs for debugging and only shares generic overall statistics with third-parties.

# Install Minecraft Plugins

## Finding plugins to use
1) First open a plugin website like [SpigotMC](https://www.spigotmc.org/), [Bukkit](https://bukkit.org/), Songoda, or BuiltByBit.
2) To find a plugin, you can use the search feature on the plugin website.
3) Make sure that the listed compatible version matches your server version.
2) After finding a plugin that you wish to use, click the "Download Now" button.

## Uploading plugins
1) First login on [https://panel.forestracks.com](https://panel.forestracks.com) and click on your server.
2) Next click the "File Manager" panel tab
3) Navigate to the `plugins` directory
4) Click the upload button and select the plugin `.jar` file from your computer or drag and drop it.

# Installing Dynmap

Dynmap is a plugin that provides a image tile website map of your Minecraft world.

1. Download the latest version of Dynmap from [here](https://www.spigotmc.org/resources/dynmap%C2%AE.274/).
2. Log into the [ForestRacks Control Panel](https://panel.forestracks.com).
3. Navigate to your server and click on the `File Manager` tab.
4. Navigate to the `plugins` folder.
5. Click on the `Upload` button and upload the Dynmap jar file.
6. Request a second port from ForestRacks support on discord or the ticket site.
7. Set the port where it says `port:` in `plugins/Dynmap/main.conf`
8. Reboot your server
9. If you use Cloudflare DNS, you can proxy your map subdomain to the Dynmap webserver port on https://dash.cloudflare.com/?to=/:account/:zone/rules/origin-rules

# Installing MinIO S3

```yml filename="docker-compose.yml" copy
version: "3"

services:
  minio:
    image: minio/minio
    command: server /data --console-address ":9001"
    environment:
      - MINIO_ROOT_USER=your_access_key_here
      - MINIO_ROOT_PASSWORD=your_secret_key_here
    volumes:
      - ./data:/data
    ports:
      - 9000:9000
      - 9001:9001

volumes:
  minio_data:
```

```bash copy
docker compose up -d
```

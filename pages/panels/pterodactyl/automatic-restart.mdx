# Configuring an automatic restart schedule

<Steps>
### Log into Panel
First login on [https://panel.forestracks.com](https://panel.forestracks.com) and click on your server. If you cannot login, follow the [Troubleshooting Guide](/games/troubleshoot/invalid-login).

### Locate Cron Schedule
1. Next click the "Schedules" panel tab
2. Click the "Create schedule" button
3. Set a schedule name
4. Input the cron timing, if you are new to cronjobs, click "Show Cheatsheet" or use the [Cronjob Creator](https://crontab.guru/)
5. Click the "New Task" button
6. Select "Send power action" from the action dropdown
7. Select "Restart" from the power action payload dropdown
8. Click "Create Task" to save

</Steps>

# Using OpenGraph

OpenGraph is commonly used on websites to provide a description, title, and image to social media platforms for embedding.

OpenGraph Images
```html copy
<meta property="og:image" content="https://example.com/logo.png" />
```

OpenGraph Title
```html copy
<meta property="og:title" content="My Website" />
```

OpenGraph Description
```html copy
<meta property="og:description" content="This is my website" />
```

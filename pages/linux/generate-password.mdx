# Generate a Password on Linux

<Tabs items={['OpenSSL', '/dev/urandom', 'Password Gen']}>
  <Tab>
    <Steps>
    ### Install OpenSSL
    ```bash copy
    apt install openssl
    ```
    ### Generate Password
    ```bash copy
    openssl rand 60 | openssl base64 -A
    ```
    </Steps>
  </Tab>
  <Tab>
    <Steps>
    ### Generate Password
    ```bash copy
    echo "$(head -c 100 /dev/urandom | LC_ALL=C tr -dc 'A-Za-z0-9!"#%&()*+,-./:;<=>?@[\]^_`{|}~')" | fold -w 32 | head -n 1
    ```
    </Steps>
  </Tab>
  <Tab>
    <Steps>
    ### Install Pwgen
    ```bash copy
    apt install pwgen
    ```
    ### Generate Password
    ```bash copy
    pwgen -s 64 1
    ```
    </Steps>
  </Tab>
</Tabs>
